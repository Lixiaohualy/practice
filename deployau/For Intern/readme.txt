
1. 程序语言
   C/C++ 或 Java, 两者一般公司都是二选一。我们比较偏向C/c++
   C++ Primer不错
   建议除了看书，需要去找习题集做(指针，引用，位操作，面向对象，模板)
2. 数据结构和算法
   链表（单，双），栈，队列，二叉树，堆，排序，字符串查找等是重点。除基本概念/用法外，经常会考如何改进
   各种变形树之类的了解一下
3. 数据库
   关系型数据库的基本概念，sql语句
4. 网络
   推荐细读TCP/IP卷一，
   实现部分一般不会考
5. 操作系统
   基本概念为 学校内的操作系统课程。文件管理，内存管理，进程管理是重点
   可能会结合具体的Linux或者Windows操作系统
6. Windows编程
   一般有实际项目经验的，这部分可以有加分效果。推荐《WINDOWS核心编程》，也有助于理解一些操作系统概念
7. 脚本语言
   常见的考Perl的正则表达式
8. 智力题
   建议没事网上搜搜做做，很多是有套路的。要设计出新的智力题并不容易。
